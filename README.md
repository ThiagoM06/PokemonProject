Pokémon Project

Projeto front-end desenvolvido com HTML, CSS e JavaScript puro, que consome a PokéAPI para buscar e exibir dados de Pokémon de forma dinâmica.

Tecnologias

HTML  
CSS  
JavaScript (ES6+)  
PokéAPI (API pública)

O que este projeto demonstra

Consumo de APIs REST utilizando fetch e async/await  
Manipulação dinâmica do DOM  
Organização clara de arquivos front-end  
Separação entre estrutura (HTML), estilo (CSS) e lógica (JavaScript)  
Boas práticas para projetos iniciais sem frameworks  

Funcionalidades

Busca de Pokémon pelo nome  
Exibição de informações básicas como nome, peso e imagem  
Atualização da interface a partir de dados externos
